 Password Strength Analyzer

 Overview
Password Strength Analyzer is a Python-based cybersecurity tool that evaluates password strength using length, complexity, and character diversity checks. It provides security recommendations, generates strong passwords, and prevents password reuse using SHA-256 hashing and SQLite database integration.

 Features
- Password Strength Analysis
- Complexity Validation
- Secure Password Generation
- Password Reuse Detection
- SHA-256 Hashing
- SQLite Database Integration
- User-Friendly GUI (Tkinter)

 Technologies Used
- Python
- Tkinter
- SQLite
- SHA-256
- Regular Expressions (Regex)
